Archie: Adventures in Self-hosting

I had an old HP ProBook 640 laying around, and wanted to give it new life as a web server. I got a free domain from noip.com, and went from there!

The server runs on Arch Linux, with a custom-built Rust backend, using the Axum framework. For the guestbook, I'm using a MySQL database to store and retrieve the entries, as well as log homepage hits. The whole setup is fully Dockerized, with the server and database each running in their own containers, orchestrated via Docker Compose.

You can see all the code used to serve this webpage on GitHub.


I hope you enjoy this little project of mine — I sure have!
